# Prerequisites

- [BioSimSpace Protein FEP tutorial](https://biosimspace.openbiosim.org/tutorials/protein_mutations.html)
- [charge-changing PFEP case-study tutorial](../mdm2-e23g-mutation/)

---

This tutorial shows how to setup an alchemical protein mutation involving a proline residue. Specifically, we will look at the Leu-to-Pro mutations in OMTKY3 to its
receptors as [detailed in this study](https://pubs.acs.org/doi/full/10.1021/acs.jctc.1c00214)

> [!Note]
> This tutorial demonstates how to run a bond-creation transformatino in SOMD2. It it does not demonstrate how to reproduce full ΔΔG values for the system above, as this is outside of the scope of the tutorial.
